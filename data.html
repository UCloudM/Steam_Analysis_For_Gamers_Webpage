Datasets

Para obtener los datos con los que funciona nuestra aplicación hemos hecho uso de 4 datasets. Todos ellos los hemos obtenido de la web kaggle.


AMD.csv

Base de datos que contiene información sobre todos los procesadores manufacturados por AMD. Esta base de datos se usa en los apartados de recomendación de los juegos en base al procesador y de recomendación de procesadores en base a un juego dado.

Entre sus columnas encontramos información como por ejemplo:
- Nombre del procesador
- Frecuencia del procesador
- Numero de núcleos
- TDP

En caso de querer descargar el dataset pulsa aquí

Intel_CPUs.csv

Base de datos que contiene información sobre todos los procesadores manufacturados por Intel. Esta base de datos se usa en los apartados de recomendación de los juegos en base al procesador y de recomendación de procesadores en base a un juego dado.

Entre sus columnas encontramos información como por ejemplo:
- Nombre del procesador
- Frecuencia del procesador
- Numero de núcleos
- TDP

En caso de querer descargar el dataset pulsa aquí

Steam.csv

Base de datos que contiene información sobre todos los juegos de Steam. Esta base de datos se usa en todas las secciones de nuestra aplicación.
Los datos de este archivo se han obtenido de la base de datos de la tienda de Steam mediante la API de SteamSpy.

Entre sus columnas encontramos información como por ejemplo:
- Nombre del juego
- AppID del juego (identificador único de este)
- Genero
- Desarrolladora del juego
- Número de valoraciones positivas
- Número de valoraciones negativas

En caso de querer descargar el dataset pulsa aquí

Steam_requirements_data.csv

Base de datos que contiene información sobre los requisitos mínimos de todos los juegos de la tienda de Steam. Esta base de datos se usa en el apartado de recomendacion de varios juegos en base a un procesador y de recomendacion de procesadores en base a un juego.
Los datos de este archivo se han obtenido de la base de datos de la tienda de Steam mediante la API de SteamSpy.

Entre sus columnas encontramos información como por ejemplo:
- AppID del juego (identificador único de este)
- Rquisitos mínimos
- Requisitos recomendados

En caso de querer descargar el dataset pulsa aquí
